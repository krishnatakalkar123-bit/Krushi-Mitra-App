Project Overview
Krushi Mitra is a comprehensive mobile solution designed to bridge the information gap between technology and traditional farming. Developed using the Flutter framework, the app aims to empower farmers in the Maharashtra region by providing real-time data that is crucial for daily agricultural decision-making.

Technical Deep-Dive
Architecture: The app follows a modular architecture for easy scalability. It uses Dart for business logic and Capacitor to bridge specialized web functionalities into the native Android environment.

API Integration: * Weather: Fetches data from the OpenWeatherMap API, providing a 5-day forecast including temperature, humidity, and precipitation alerts.

Market Mandi Rates: Integrated with agricultural portals to provide daily price updates (Minimum, Maximum, and Modal prices) for various commodities across APMCs in Maharashtra (e.g., Pune, Latur, Jalna).

User Interface (UI): Focused on a 'Minimalist & High-Contrast' design. Since the target users are farmers who might use the app in bright sunlight, we used bold fonts and clear icons for better visibility.

Problem Statement & Solution
Problem: Farmers often rely on middlemen or outdated news for market prices, leading to financial losses.

Solution: Krushi Mitra provides direct, real-time access to official government data and global weather patterns, ensuring the farmer is always one step ahead.

Future Roadmap
AI Disease Detection: Implementing a feature where farmers can upload a photo of a leaf to identify crop diseases.

Multilingual Support: Expanding the app to support full Marathi and Hindi localizations for better accessibility.

Expert Consultation: A chat module to connect farmers directly with agricultural experts.
