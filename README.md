Project Name: Krushi App

Description:
This is a Flutter-based application for farmers.
It provides features like crop information, weather updates, etc.

Technology Used:
- Flutter
- Dart

How to Run:
1. Download project
2. Run flutter pub get
3. Run flutter run

4. Note:
The APK file may show a warning on Google Drive because it is a large executable file.
Click on "Download anyway" to proceed. The file is safe and developed for academic purposes.
